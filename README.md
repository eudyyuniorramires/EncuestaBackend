# EncuestaBackend

Un backend desarrollado en C# y .NET para la gestión de encuestas. 

## Estructura del Proyecto

La solución está dividida en varias capas:
- **Data**: Capa de acceso a datos.
- **Entidades**: Modelos y entidades del dominio.
- **Logica**: Lógica de negocio de la aplicación.
- **Presentacion**: Capa de presentación (API).

## Tecnologías
- C#
- .NET

## Cómo ejecutar el proyecto
1. Clona el repositorio.
2. Abre la solución `EncuestaBackend.sln` en tu entorno de desarrollo (ej. Visual Studio).
3. Compila la solución y ejecuta el proyecto de presentación.
